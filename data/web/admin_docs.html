Admin Documentation

Admin Responsibilities

Admins are responsible for maintaining a clean and current knowledge base inside each workspace.

Document Management

Admins can upload new files, delete outdated files, assign categories, and rebuild the index after major content changes.

Index Rebuild

Rebuilding the index ensures that newly uploaded content and deleted files are correctly reflected in retrieval results.

Analytics Review

Admins can review the most asked questions, failed queries, and usage patterns to identify gaps in documentation coverage.

Access Control

Only authorized roles should be allowed to delete documents, manage categories, or change workspace-level settings.
