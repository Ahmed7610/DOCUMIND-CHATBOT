API Overview

Why API Integration Matters

Some user questions cannot be answered from documents alone. These cases require live data from tools or APIs.

Supported Agent Use Cases

  • Checking subscription status
  • Tracking support ticket status
  • Retrieving order or account-specific data

Document Questions vs Live Questions

Questions about policies, setup steps, plan features, and documented workflows should be answered from indexed content.

Questions about live user status or account-specific information should be answered using connected APIs or simulated tools.

Example Tools

  • get_subscription_status(user_id)
  • get_ticket_status(ticket_id)

Expected Agent Behavior

The assistant should decide whether a user question requires retrieval from documents or a call to a live tool.
